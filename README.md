# 糖尿病个人健康管理系统 java516

## 项目概述

糖尿病个人健康管理系统是基于SSM（Spring、SpringMVC、MyBatis）框架开发的，旨在帮助糖尿病患者有效管理个人健康信息。该系统适用于计算机相关专业人员及Java学习者的实战练习。

## 技术栈

- **后端技术**：Spring、SpringMVC、MyBatis
- **前端技术**：JSP、HTML、CSS、JavaScript、jQuery
- **数据库**：MySQL
- **系统架构**：MVC、B/S
- **开发环境**：JDK、Eclipse或IDEA、Tomcat、MySQL

## 功能模块

系统包含两种角色：用户、管理员，主要分为前台和后台两大模块。

### 前台功能模块

- 网站首页
- 新闻公告
- 健康知识
- 在线预约体检
- 用户注册与登录
- 个人中心
- 我的预约
- 个人健康日常数据
- 日常数据图表
- 修改个人信息
- 修改密码
- 意见反馈

### 后台功能模块

- 系统登录
- 管理员信息管理
- 用户信息管理
- 网站栏目管理
- 网站公告管理
- 日常数据管理
- 健康建议管理
- 体检预约管理
- 意见反馈管理
- 日志信息管理

## 系统角色

- 用户：管理个人健康信息，进行在线预约体检等。
- 管理员：负责系统的日常管理和维护。

## 运行环境
